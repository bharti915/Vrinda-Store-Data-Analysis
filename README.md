# Vrinda Store Data Analysis

## Project Overview
This project focuses on analyzing the sales data of Vrinda Store using Excel. 
The goal of this project is to extract meaningful insights from the dataset and understand sales trends, customer behavior, and product performance.

## Objectives
- Analyze monthly sales performance
- Identify top-selling products
- Understand customer demographics
- Compare sales by gender and age group
- Identify the best sales channels

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts and Graphs
- Data Cleaning Techniques

## Dataset
The dataset contains information such as:
- Order ID
- Customer Gender
- Age
- Product Category
- Sales Amount
- Order Date
- Sales Channel

## Key Insights
- Women customers contributed more to total sales.
- Certain months showed higher sales trends.
- Some sales channels performed better than others.
- Specific product categories generated the highest revenue.

## Project Files
- Vrinda Store Data Analysis.xlsx → Contains the complete data analysis.

## Conclusion
This analysis helps understand customer buying behavior and sales performance. Businesses can use these insights to improve marketing strategies and increase sales.
